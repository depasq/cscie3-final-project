Joseph DePasquale - CSCI E-3 Final Project

My final project uses javascript to read from a JSON-formatted database of Chandra press releases to serve up a random press release. The code also uses the WorldWide Telescope API to show location of the object on the sky.


Explore the X-ray Universe - Chandra Press Releases in Context

Click "Surprise Me!" to load a random press release below. Content of the release is displayed on the right, while a contextual view on the optical sky is provided in the portal on the left via WorldWide Telescope. The thumbnail image is a link that takes you to the press release web page.
